Memodify

Keep noting daily, personally and securely

Taking notes is extremely personal, private, secret, and confidential. Without thinking, people may accidentally write down things that they don't want others to see, such as passwords or the number of their total assets. Are you sure that no one can see the application you are writing in?

With Memodify, your data is stored locally. Memodify stores your data locally and does not have the ability to share it with others. There are no plug-ins, nothing. You can write whatever you want.

What's this?

Memodify is a Markdown editor.

  • Powerful preview with MDX
  • Developer friendly
  • Privacy-considered
  • Standalone & In-brower

Feature

Not rich features, but enough

Simple is best!

Command Palette

Command palette is an awesome UI. You can quickly invoke the palette whenever you want. In addition, a note palette is supported.

Dark mode

Dark mode is supported for sure. You can change the mode whenever you want throught the command palette.

Power of MDX

Memodify is a note-taking application supporting MDX. “MDX is an authorable format that lets you seamlessly write JSX in your Markdown documents.“ You can see how it's rendered with live preview. (Experimental)

Theme

Several color schemes are pre-defined. You can change your active theme anytime.

Organize notes

Easily organize your notes with label which is a popular function as metadata. You can star, like, complete and trash notes, then you can filter with the statues.

More features

There are some features you are familiar with.

Export and import

You can export all your notes as a plain JSON file even for In-browser Memodify. You can stop using Memodify anytime and transfer your data somewhere.
Importing works, too, certainly.

Daily Noting

You can continue editing a single note so that you don't need creating a new file, thinking filename or title, etc. It's automatically categorized, classified and grouped.
(In progress)

All in local

All your data is in your local machine. No your data go through the Internet unintentionally. No sync, no share and no publish. Memodify never sends your data to any remote hosts without your directions.

Images by DnD

You can drag & drop images onto notes of Memodify to attach them. They are referrable in the note. The attached images are exported with notes to a JSON in base64.

GFM

Memodify supports GitHub Flavored Markdown so called GFM. You can write listing, tables, quoting, Math expression, etc as GitHub comments though it's a subset.

Zen mode

“Zen“ mode allows you to concentrate on a task with a simplified UI. In memodify The simplest layout of Memodify consists of a few text fields.

Encryption

Memodify allows you to encrypt note body. It is possible on a per note basis. This feature can be useful in situations where some notes need to be confidential. (Experimental)

Undo/redo

Undo and redo are supported with the exception of a few features like empty-trash, even though in-browser mode.

Chart.js & Mermaid

Chart.js is available. You can put some types of charts in your note and see them in the preview.Mermaid is available, too. Memodify supports them as builtin components.
