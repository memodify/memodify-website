# Settings

## config.yaml

You can put a configuration file to give some values like secrets to the Memodify application.
`~/.config/memodify/config.yaml`

```yaml
github:
  # https://github.com/settings/tokens
  token: "ghp_************************************"

openai:
  # https://platform.openai.com/account/api-keys
  api_key: "sk-************************************************"
```

### Supported items
| Item  | Description |
| ------------- | ------------- |
| `github.token`  | Used to retrieve <a href="#replaceurlprivaterepository">issue title in private repositories</a>. |
| `openai.api_key`  | Used for ChatGPT feature.<br/> <a href="/blog/chatgpt-integration/">This blog entry</a> is for an experimental feature for ChatGPT. |
